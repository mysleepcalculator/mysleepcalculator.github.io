## 🚀 Deploy on GitHub Pages

1. Create a repo named **mysleepcalculator.github.io** (under the `mysleepcalculator` account – yours).
2. Upload all files from this folder to the **root** of that repo.
3. Repo → Settings → Pages → Source: **Deploy from a branch** → Branch: `main` / root → Save.
4. Site goes live at https://mysleepcalculator.github.io/ within ~1 minute.
