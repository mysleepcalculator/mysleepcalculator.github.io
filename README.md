## 🚀 Deploy to GitHub Pages

1. Create a **new GitHub account/organisation** named `mysleepcalculator`
   (so the URL becomes `https://mysleepcalculator.github.io/`).
2. Create a new repository: **`mysleepcalculator.github.io`** (this exact name).
3. Upload all files in this folder to the repo root (do NOT put them in a sub-folder).
4. Go to **Settings → Pages**:
   - Source: `Deploy from a branch`
   - Branch: `main`  /  folder: `/ (root)`
   - Save.
5. Wait 1–2 minutes. Your site is live at:
   **https://mysleepcalculator.github.io/**
